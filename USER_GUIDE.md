# ProjectManager 用户操作手册

## 快速开始
- 安装：双击 `ProjectManager.dmg`，将 App 拖入“应用程序”，直接启动。
- 开发调试（可选）：在仓库根目录运行 `./build.sh` 或 `swift run ProjectManager`。
- 界面概览：左侧为“管理目录 + 标签”，右侧为“项目列表、搜索与排序”。

## 添加项目
- 添加工作目录（推荐）：侧边栏“添加目录…” → “添加工作目录（扫描项目）”，会把所选目录的一层子目录识别为项目。
- 直接添加为项目：侧边栏“添加目录…” → “直接添加为项目”，可一次选择多个文件夹加入。
- 扫描子目录并添加：选择任意文件夹，将其所有直接子目录批量添加为项目。
- 拖拽添加：把文件夹拖拽到“管理目录”区域，即可加入监视目录。
- 管理监视目录：在“管理目录”菜单中可移除目录或“在访达中显示”。

## 浏览与筛选
- 搜索：顶部搜索框按项目名或路径实时过滤。
- 标签筛选：点击左侧任一标签按该标签过滤；“全部”显示全部项目；“没有标签”仅显示未打标签的项目。
- 目录筛选：选中某个监视目录，仅显示该目录下的项目。
- 排序：支持按修改时间（升/降）或 Git 提交数排序。

## 标签管理
- 新建标签：侧边栏“标签”标题右侧“+”创建，可设置颜色。
- 重命名标签：在标签行的更多操作中选择“重命名”。
- 给项目打标签：
  - 在项目卡片点击“编辑标签”添加/移除；或
  - 将项目卡片拖拽到侧边栏某个标签上快速打标签。
- 移除项目标签：在“编辑标签”弹窗中取消勾选，或点击项目卡片标签上的“×”（如提供）。

## 打开项目
- 快捷操作：项目卡片提供“在编辑器打开”“在访达显示”“在终端打开”“复制路径/项目信息”等。
- 打开方式菜单：从“打开方式”选择 Cursor、VS Code、Trae 等编辑器。
- 首选编辑器：在“偏好设置”（Cmd + ,）设置默认编辑器，应用会自动检测已安装编辑器。

## 刷新与导入
- 刷新项目：侧边栏“刷新与重载” → “刷新项目”。
- 导入备份：在“数据导入”页面选择 JSON 备份文件，支持“合并/替换/跳过已存在”与冲突处理策略，导入后显示结果与统计。

## 数据与限制
- 本地数据：应用将项目与标签缓存到本地以提升加载速度。
- 系统标签：当前版本不会把应用内标签写回 Finder 文件标签（系统元数据）。
