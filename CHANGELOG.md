# 更新日志

本文件记录了 ProjectManager 的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]

### 新增
- 项目卡片标签点击筛选功能
- 标签点击视觉反馈效果（悬停和点击动画）
- 完整的标签点击回调链路实现

### 修复
- TagView 点击事件被项目卡片拦截的问题
- 标签点击后正确触发侧边栏筛选逻辑

### 改进
- 重构 TagView 为 Button 实现，提高点击可靠性
- 增强用户交互体验和视觉反馈

## [1.0.0] - 2024-12-XX

### 新增
- 🎉 首次发布 ProjectManager
- 智能项目发现和自动扫描功能
- macOS 文件标签系统深度集成
- 多编辑器支持（Cursor、VSCode、Trae AI、自定义编辑器）
- Git 信息集成（提交次数、分支信息）
- 现代化 SwiftUI 界面设计
- 响应式项目网格布局
- 标签管理和可视化筛选
- 项目拖拽和批量操作
- 实时目录监控和增量更新
- 暗色模式支持

### 核心功能
- **项目管理**: 自动发现、标签分类、搜索筛选
- **编辑器集成**: 智能检测、一键打开、自定义配置
- **Git 集成**: 提交统计、分支管理、状态展示
- **用户体验**: 直观界面、快捷操作、平滑动画

### 技术特性
- Swift 5.7+ / SwiftUI 原生开发
- MVVM 架构模式
- 纯函数业务逻辑设计
- macOS 12.0+ 支持
- 高性能增量更新机制

## 发布节奏

我们遵循语义化版本控制：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向后兼容的功能性新增
- **修订号**：向后兼容的问题修正

每个版本都会包含详细的升级指南和迁移说明（如果需要）。
